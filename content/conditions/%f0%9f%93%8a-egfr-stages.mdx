        <summary class="head"><span class="badge">1</span> What is eGFR? <span class="chev">▾</span></summary>
        <div class="content">
          <p><strong>eGFR</strong> (estimated glomerular filtration rate) is a lab estimate of how well your kidneys filter blood. It’s calculated from blood creatinine plus age and sex; the most current equations do <em>not</em> use race. <strong>Trends</strong> over months matter more than a single number.</p>
          <ul>
            <li><strong>Higher eGFR is better.</strong> Lower numbers mean less filtering.</li>
            <li><strong>Albumin in urine</strong> (protein leak) adds risk information beyond eGFR alone.</li>
            <li><strong>Hydration, illness, and some medicines</strong> can temporarily change eGFR.</li>
          </ul>

        <div class="content">
          <p><strong>Urine albumin-to-creatinine ratio (ACR)</strong> checks if protein is leaking into urine. More protein typically means higher kidney and heart risk.</p>

              <h3>Trends matter</h3>
              <ul>
                <li>Compare <strong>several months</strong>, not one day.</li>
                <li>Illness, dehydration, and some medicines can temporarily lower eGFR.</li>

      <!-- 6 Improve kidney & heart protection -->
      <details class="section">
        <summary class="head"><span class="badge">6</span> Protecting kidneys & heart (daily steps) <span class="chev">▾</span></summary>
        <div class="content">
          <div class="three">
            <div class="box">
              <h3>Blood pressure</h3>
              <ul>
                <li>Accurate home checks, calm posture, same times daily when adjusting treatment.</li>
                <li>Share averages (morning/evening over several days).</li>
              </ul>
            </div>
            <div class="box">
              <h3>Salt & meals</h3>
              <ul>
                <li>Taste first; use herbs/lemon instead of extra salt.</li>
                <li>Limit processed meats, instant soups, sauces.</li>
                <li>Ask about potassium/phosphorus limits for your stage.</li>
              </ul>
            </div>
            <div class="box">
              <h3>Movement & sleep</h3>
              <ul>
                <li>Walk 5–10 minutes after meals; add gentle strength and balance.</li>
                <li>Consistent bedtime; cool, dark room; safe night light.</li>
              </ul>

      <!-- 7 Meds & what to avoid -->
      <details class="section">
        <summary class="head"><span class="badge">7</span> Medicines & what to avoid <span class="chev">▾</span></summary>
        <div class="content">
          <div class="two">
            <div>
              <h3>Often used (individualized)</h3>
              <ul>
                <li>ACE inhibitor or ARB (especially with albumin leak)</li>
                <li>SGLT2 inhibitor (for many with diabetes; sometimes without)</li>
                <li>Diuretics (“water pills”) for salt/fluid control</li>
                <li>Statins to reduce heart risk</li>
              </ul>
            </div>
            <div class="note warn">
              <strong>Use with caution / avoid unless cleared:</strong>
              <ul>
                <li>Many <strong>NSAIDs pain pills</strong> (over-the-counter) can worsen kidney function and raise BP</li>
                <li>Herbal products with unknown kidney effects</li>
                <li>Contrast dye scans without kidney-safe planning</li>
              </ul>
            </div>
          </div>
          <p class="muted">Never start/stop medicines on your own—doses depend on eGFR, potassium, and other conditions.</p>
        </div>

          <div class="box" itemscope itemprop="mainEntity" itemtype="https://schema.org/Question">
            <h3 itemprop="name">What eGFR is considered “kidney failure”?</h3>
            <div itemscope itemprop="acceptedAnswer" itemtype="https://schema.org/Answer">
              <p itemprop="text">eGFR &lt; 15 (G5) is considered kidney failure. Planning with a kidney specialist usually starts earlier (G4) so decisions are calm and personalized.</p>
            </div>
          </div>
          <div class="box" itemscope itemprop="mainEntity" itemtype="https://schema.org/Question">
            <h3 itemprop="name">Can dehydration lower eGFR temporarily?</h3>
            <div itemscope itemprop="acceptedAnswer" itemtype="https://schema.org/Answer">
              <p itemprop="text">Yes. Illness and dehydration can temporarily lower eGFR. Your clinician may repeat labs when you’re well and hydrated.</p>
            </div>
          </div>
          <div class="box" itemscope itemprop="mainEntity" itemtype="https://schema.org/Question">
            <h3 itemprop="name">Does muscle mass affect creatinine?</h3>
            <div itemscope itemprop="acceptedAnswer" itemtype="https://schema.org/Answer">
              <p itemprop="text">Yes. Low muscle mass can make creatinine lower and eGFR seem higher than true filtration. Clinicians interpret results in context.</p>

    const val = document.getElementById('egfrVal');
    const stage = document.getElementById('egfrStage');

    function toMgDl(v, units){
      const x = parseFloat(v);
      if(!isFinite(x) || x<=0) return NaN;
      return units==='umol' ? (x/88.4) : x;
    }

    function estimateEgfr(age, sex, scrMgDl){
      // Very simplified race-free CKD-EPI–style estimate (educational!)
      const k = (sex==='female') ? 0.7 : 0.9;
      const a = (sex==='female') ? -0.241 : -0.302;
      const alpha = (sex==='female') ? -0.241 : -0.302;
      const minScr = Math.min(scrMgDl/k, 1);
      const maxScr = Math.max(scrMgDl/k, 1);
      // Base coefficient ~142, female factor ~1.012 (approx); age factor 0.993^age
      const femaleFactor = (sex==='female') ? 1.012 : 1.0;
      const egfr = 142 * Math.pow(minScr, alpha) * Math.pow(maxScr, -1.200) * Math.pow(0.993, age) * femaleFactor;
      return egfr;
    }

    function stageFromEgfr(g){
      if(g>=90) return 'G1 (normal/high)';
      if(g>=60) return 'G2 (mildly decreased)';
      if(g>=45) return 'G3a (mild–moderate)';
      if(g>=30) return 'G3b (moderate–severe)';
      if(g>=15) return 'G4 (severely decreased)';
      return 'G5 (kidney failure)';
    }

    function go(){
      const age = parseInt(ageEl.value,10);
      const sex = sexEl.value;
      const scr = toMgDl(scrEl.value, unitsEl.value);
      if(!isFinite(age)||age<1||age>120||!isFinite(scr)){
        out.style.display='none';
        return;
      }
      const g = estimateEgfr(age, sex, scr);
      const gRound = Math.max(0, Math.round(g));
      val.textContent = 'eGFR: ' + gRound + ' mL/min/1.73m²';
      stage.textContent = 'Stage: ' + stageFromEgfr(gRound);
      out.style.display='flex';
    }

    btn && btn.addEventListener('click', go);
    clr && clr.addEventListener('click', ()=>{
      ageEl.value='70'; sexEl.value='male'; scrEl.value='1.2'; unitsEl.value='mgdl';
      out.style.display='none';
    });
  })();
